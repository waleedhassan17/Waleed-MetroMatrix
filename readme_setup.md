# Authentication Module Setup Instructions

## 🔐 Sensitive Files Not Included in Git

The following files contain sensitive credentials and are **NOT** included in this repository:

### 1. `google-services.json`
- **Where to get it:** Firebase Console → Project Settings → Your Android App → Download google-services.json
- **Location:** Place in project root directory
- **Required for:** Android Google Sign-In

### 2. `credentials.json` and `credentials/` folder
- **Where to get it:** Generated by EAS: `eas credentials`
- **Location:** Project root
- **Required for:** Android app signing

### 3. Firebase Configuration
Update `firebaseConfig.js` with your Firebase project credentials:
```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT.firebasestorage.app",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

### 4. Authentication Provider IDs
Update `authConfig.js`:
```javascript
export const AUTH_CONFIG = {
  GOOGLE_WEB_CLIENT_ID: 'your-web-client-id.apps.googleusercontent.com',
  GOOGLE_ANDROID_CLIENT_ID: 'your-android-client-id.apps.googleusercontent.com',
  FACEBOOK_APP_ID: 'your-facebook-app-id',
};
```

### 5. Facebook Configuration in app.json
```json
{
  "plugins": [
    [
      "react-native-fbsdk-next",
      {
        "appID": "YOUR_FACEBOOK_APP_ID",
        "clientToken": "YOUR_FACEBOOK_CLIENT_TOKEN"
      }
    ]
  ]
}
```

---

## 🚀 Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Create the sensitive files listed above
4. Update all credentials in config files
5. Run: `npm start` (for web) or `eas build` (for Android)

---

## 📝 Notes

- All sensitive credentials are kept private for security
- Each team member needs to configure their own Firebase/Google/Facebook projects
- For production, use environment variables or secure credential management
